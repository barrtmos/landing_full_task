# Landing Full Task

Внешний лендинг с видео, таймером, формой захвата лида и трекингом через Facebook Pixel и Binom.

---

## Структура файлов

```
landing_full_task/
├── index.html   — разметка страницы
├── style.css    — стили
├── app.js       — вся логика
└── video.mp4    — промо-видео
```

---

## Как работает

Лендинг принимает трекинговые параметры из URL при загрузке страницы и использует их на протяжении всего флоу.

### Видео и таймер

Пользователь должен просмотреть определённое количество секунд видео. Таймер отсчитывает только реально просмотренное время — пауза и перемотка учитываются корректно. После достижения порога таймер исчезает и появляется форма.

### События при разблокировке формы

При появлении формы стреляют события в Facebook Pixel и Binom.

### Форма

Поля: имя, фамилия, email, телефон. Данные валидируются на фронте перед отправкой. Вместе с данными формы передаются трекинговые параметры из URL.

### Обработка ответа API

- Успех — форма скрывается, показывается сообщение, стреляют события конверсии в Facebook и Binom
- Ошибка валидации — показываются ошибки под полями
- Ошибка сервера / сеть — показывается общее сообщение об ошибке

---

### Binom трекинг

Используется LP Pixel — скрипт встроен в `<head>` и регистрирует клик в Binom при загрузке страницы без редиректа. При разблокировке формы и успешной отправке стреляют события через `BPixelJS`.
